# TFM_Ebis
TFM desarrollado por Sergio Polo Garcia para el master Ingeniería y Desarrollo de Soluciones de IA Generativa de ebis Bussines Techschool.

El trabajo cosiste en un sistema de Speech Analytics con 2 verticales, una de procesamiento/indexacion de transcripciones de llamadas y otra enfocada a la disponibilizacion del dato utilizando consultas en lenguaje natural.
